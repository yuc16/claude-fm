这条消息是七月十一号,Anthropic 正式发布了 Claude 二代模型,也就是 Claude 二点零。这次发布有几个看点。第一,它第一次有了面向公众的测试网站,叫 claude dot ai,美国和英国的用户当天就能直接上手体验聊天,不再只是开发者通过 API 才能用。第二,在性能上,Claude 二代相比上一代 Claude 一点三有明显提升,官方举了个例子,在律师资格考试的选择题部分,得分从百分之七十三提高到了百分之七十六点五。在 GRE 的阅读和写作部分,Claude 二代的得分超过了九成申请研究生院的学生,数学推理部分则和中位数水平的申请者差不多。第三,也是我个人觉得对开发者最实用的一点,上下文窗口大幅拉长,单次输入最多可以塞进十万 token,相当于几百页的技术文档,甚至一整本书都能一次性丢给它处理,输出方面也支持一次性生成几千 token 的长文档,从备忘录到书信到故事都可以。第四,官方还特别强调,这一代在安全性上做了优化,用户反馈说 Claude 更不容易给出有害内容,而且记忆能力更长,对话体验更连贯。最后,商业上的一个细节是,企业用户通过 API 调用 Claude 二代,价格和上一代 Claude 一点三保持一致,没有涨价。

这条消息之所以值得放进 Anthropic 的发展脉络里看,是因为它标志着 Anthropic 第一次把模型能力和面向消费者的产品体验放在一起推出,而不只是闷头做 API 业务,这也为后面 Claude 系列逐步走向大众市场打下了基础。

这周的快讯就到这里,核心就是记住一句话,Claude 二代来了,上下文变长了,网站也对外开放了。我们下期再见。
