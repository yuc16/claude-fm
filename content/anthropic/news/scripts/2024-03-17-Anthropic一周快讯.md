先说第一条,大概是三月十九号,Anthropic 宣布 Claude 三的 Haiku 和 Sonnet 两个模型在 Google Cloud 的 Vertex AI 平台上正式全面可用了。这个事情值得说两句,因为对企业客户来说意义不小,他们可以直接在自己已经在用的 Google Cloud 环境里调用 Claude,数据不用挪到别的地方,既简化了数据治理,又能更方便地管控权限,降低了运营复杂度。这其实是 Anthropic 持续推进多云分发策略的一部分,之前已经上了 AWS 的 Bedrock,现在再补上 Google Cloud 这一块,等于把企业客户能接触到 Claude 的渠道又扩大了一圈。官方还提到一个案例,问答平台 Quora 旗下的 Poe 应用已经在用 Claude,用户反馈说它在创意写作和图片理解这些场景里表现不错,回答也比较详细。

接下来是这周第二条,大概是三月二十号,Anthropic 联合 AWS 和埃森哲三方一起宣布了一项合作,目标是帮企业把生成式 AI 从想法落地成真正能用的产品,尤其是金融医疗这类对准确性和数据安全要求很高的监管行业。这条偏公司动态,一句话讲完重点就是,埃森哲会培训超过一千四百名工程师成为 Anthropic 模型在 AWS 上的专家,帮客户用自己的数据去微调模型,提升在特定行业里的表现。材料里提到一个已经落地的例子,华盛顿特区的卫生部门用 Claude 通过 Amazon Bedrock 做了一个支持中英文双语的问答机器人,叫 Knowledge Assist,居民和员工可以直接用自然语言问健康服务相关的问题。

这一周整体来看,Anthropic 没有发新模型,更多是在巩固 Claude 三在各大云平台和企业服务体系里的落地,我个人觉得 Vertex AI 上线这条更值得关注一点,因为它直接关系到开发者能从哪些渠道用上 Claude。本期快讯就到这里,我们下期再见。
