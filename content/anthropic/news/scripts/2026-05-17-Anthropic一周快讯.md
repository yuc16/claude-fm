大家好,欢迎收听 Claude FM,这里是 Anthropic 一周快讯。本期我们覆盖的是五月十七号到五月二十三号这一周,一共三条消息,我们按时间顺序来说。

第一条,五月十八号,Anthropic 宣布收购了一家叫 Stainless 的公司。Stainless 是做什么的呢,简单说就是帮各家公司把 API 自动生成成各种编程语言的 SDK,还有命令行工具和 MCP 服务器,也就是让开发者和智能体能够方便调用接口的那一整套连接工具。其实从 Claude API 上线最早期开始,官方的各个语言 SDK 就是 Stainless 帮忙生成的,这次干脆把整个团队收进了 Anthropic。这条消息我个人觉得挺值得注意,因为它呼应了 Anthropic 现在的一个大方向,就是从单纯做模型转向做能够真正动手干活的智能体,而智能体的能力很大程度上取决于它能连接多少工具和数据源,把负责这块的团队直接收购过来,说明 Anthropic 在往这个方向加重投入。

接下来,同样是五月十九号,Anthropic 宣布和毕马威也就是 KPMG 达成了一项全球战略联盟。毕马威是全球最大的审计、税务、法律和咨询服务机构之一,业务覆盖一百三十八个国家和地区。这次合作里,Claude 会被嵌入到毕马威员工和客户日常办公使用的 Digital Gateway 系统里,先从税务和法律相关的客户工具开始落地,而且毕马威全球超过二十七万六千名员工都将获得 Claude 的使用权限。另外 Anthropic 还把毕马威定为私募股权领域的优先合作伙伴,双方会一起给私募股权的投资组合公司开发新产品。这条算是比较典型的企业合作公司动态,一句话带过就好。

最后一条,也是五月十九号,Anthropic 发布了一篇文章,讲他们这几个月在做的一件事,就是和各种持有不同世界观的群体展开对话,第一轮找的是来自十五个以上宗教和跨文化背景的学者、神职人员、哲学家和伦理学家。这么做的原因是,Anthropic 觉得光靠技术上的对齐、可解释性这些工作还不够,因为 AI 已经实实在在影响着很多人的生活,所以在制定 Claude 的价值观框架、也就是所谓的 Claude 宪法的时候,需要听取哲学家、神职人员、心理学家这些在相关问题上已经做了大量思考的人的意见。这条我觉得也挺有意思,它不是技术发布,但反映出 Anthropic 在 AI 安全和价值观对齐这件事上,正在尝试把视角从单纯的工程团队拓展到更广泛的社会群体,这也是理解他们怎么看待负责任 AI 这个问题的一个新角度。

好,这周的快讯就是这三条,一个收购、一个大型企业合作、一次关于价值观的跨界对话。我个人觉得最值得记一下的还是收购 Stainless 这件事,因为它透露出的信号是 Anthropic 接下来会更加重视智能体和外部工具之间的连接能力。本期就到这里,我们下期再见。
