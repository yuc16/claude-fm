## Script

大家好,欢迎收听 Claude FM 的 Anthropic 一周快讯,这期我们看的是十月二十号到十月二十六号这一周,本周一共两条消息,但其实是同一天发布的两件大事,所以这期内容会比较集中,我们慢慢说。

这两条消息都发生在十月二十二号。第一件事是 Anthropic 发布了升级版的 Claude 三点五 Sonnet,还有一个全新的模型,叫 Claude 三点五 Haiku。升级版的 Sonnet 在各项能力上都有全面提升,尤其是写代码这个方向,本来 Sonnet 就是这个领域里比较领先的模型,这次提升幅度还挺明显。而 Claude 三点五 Haiku 这边更值得说一说,它在很多评测上的表现已经追平了上一代的旗舰模型 Claude 三 Opus,但速度跟上一代 Haiku 差不多。也就是说,一个原本是小模型定位的 Haiku,现在做到了上一代大模型的水平,这对追求性价比和响应速度的开发者来说,是个挺实用的升级,意味着很多原来要用大模型才能做的任务,现在用更快更便宜的小模型也能搞定。

第二件事,也是这一周里最值得关注的一条,Anthropic 同时推出了一个全新的能力,叫 computer use,中文可以理解为让 Claude 操作电脑。简单说就是开发者可以让 Claude 像人一样使用电脑,通过看屏幕截图、移动鼠标、点击按钮、输入文字来完成任务。Anthropic 强调 Claude 三点五 Sonnet 是第一个把这种操作电脑能力开放公测的前沿模型。官方也很诚实地说,这个能力目前还是实验阶段,有时候会显得笨拙,也容易出错,所以先开放出来收集开发者的反馈,预期后续会快速迭代。已经有不少公司在尝鲜了,像 Asana、Canva、Cognition、DoorDash、Replit、The Browser Company,都在用这个能力去完成一些需要几十步甚至上百步操作才能搞定的复杂任务。

Anthropic 同时还发了一篇博客,专门讲这个 computer use 模型背后的研发思路和安全考量,核心观点是,现在大量的工作都是在电脑上完成的,如果 AI 能像人一样直接操作各种软件,而不是只能通过专门定制的工具去交互,那能解锁的应用场景会非常多。这也被 Anthropic 看作是继复杂推理、图像理解之后,AI 发展的下一个重要里程碑。

我个人觉得,这一周虽然只有一条核心新闻,但信息密度挺高,一边是模型本身能力和性价比的提升,一边是交互方式上的突破,两者放在一起,其实指向的是同一个方向,就是让 Claude 不仅更聪明,还能更直接地帮你把活干完。这期就到这里,我们下周再聊。
