大家好,欢迎收听 Claude FM,这里是 Anthropic 一周快讯。本期我们覆盖的是二月二十三号到三月一号这一周,这周一共有四条消息值得说一说,其中有一条是重磅的模型发布,我们多花点时间讲。

先说最重要的一条。二月二十四号,Anthropic 发布了 Claude 三点七 Sonnet,官方说这是他们目前最聪明的模型,也是市场上第一款混合推理模型。什么叫混合推理呢,就是这一个模型既能做到几乎即时响应,也能切换到一种叫扩展思考的模式,让模型把一步一步的思考过程展现给用户看,而且通过 API 调用的开发者还可以精细控制模型思考的时长,等于是把快思考和慢思考揉进了同一个大脑里,而不是像有些厂商那样用两个独立的模型分别处理。这次升级在编程和前端网页开发上的提升尤其明显。配合这次发布,Anthropic 还推出了一个命令行工具,叫 Claude Code,目前是限量研究预览阶段,核心思路是让开发者直接在终端里把比较重的工程任务交给 Claude 去做,这其实是 Anthropic 在智能体编程方向上一个比较实质的落地动作。新模型已经全面上线到 Claude 的免费版、专业版、团队版和企业版,也同步上了 Claude 开发者平台、亚马逊 Bedrock 和谷歌云 Vertex AI,价格和上一代保持一致,没有因为推理能力升级而涨价,这点对开发者还是比较友好的。

接下来一条相对轻一些。二月二十六号,Anthropic 宣布 Claude 模型开始为亚马逊的 Alexa Plus 提供支持,这是双方持续合作的一部分,Alexa Plus 通过 Bedrock 来调用 Claude,会先在美国小范围上线,之后逐步扩大覆盖,这条主要是公司层面的合作进展,了解一下就行。

另外这周,二月二十七号,Anthropic 上线了一个叫 Transparency Hub 的透明度中心,公开了一批关于平台安全和治理的具体指标,包括封禁账号数量、申诉和申诉被驳回的情况,以及对未成年人保护相关机构的报告数据等等。这个动作本身知识含量不算特别高,但放在整个行业脉络里看,值得注意的地方在于,随着前沿模型能力越来越强,监管框架也在演进,Anthropic 选择主动把内部的安全测试方法、风险评估流程这些东西公开化、标准化,这是一种在监管和公众信任压力下,头部 AI 公司主动做自我约束和披露的姿态,这个趋势我个人觉得后面其他厂商可能也会跟进。

最后一条,二月二十八号,Anthropic 参与了美国能源部主办的首届千名科学家 AI Jam 活动,联合多个国家实验室的科学家一起评测前沿模型在科研和国家安全相关场景下的表现,本周刚刚发布的 Claude 三点七 Sonnet 就被拿去做了实测,科学家们会用真实的科研问题去检验模型在文献检索、假设生成、实验规划这些环节上的能力。这条更多是体现 Anthropic 在拓展模型应用场景上的布局,简单了解就好。

总的来说,这一周 Anthropic 的核心动作还是发布 Claude 三点七 Sonnet 和 Claude Code 这两件事,混合推理加上命令行智能编程,基本是把这一周的分量都集中在了模型能力本身上,剩下几条更多是合作和治理层面的补充。本期就到这里,我们下期再见。
