第一条是一月十一号。Anthropic 宣布扩展 Claude 在医疗和生命科学领域的能力,推出了一个叫 Claude for Healthcare 的新产品线。简单说,这是一套面向医疗机构、保险方,还有医疗科技公司的工具,符合 HIPAA 也就是美国医疗隐私合规的要求,可以让 Claude 真正用到看病、理赔这些场景里去。同时他们还在帮普通人理解自己的健康数据。除此之外,生命科学这一侧也在加强,把 Claude 接到更多科研平台上,从临床试验管理到药品监管流程都能帮上忙。这条信息背后比较有意思的一点是,他们专门提到 Claude Opus 四点五这个模型,在医疗和科研类的模拟任务上有明显进步,而且在诚实性评测上,也就是减少胡编乱造这一块,也有实质提升。这说明 Anthropic 推医疗产品不是单纯堆功能,而是模型本身的可靠性已经支撑到了能往严肃场景去用的程度,这对关注大模型在垂直行业落地的朋友来说,是个值得记一下的信号。

接下来是一月十三号,Anthropic 宣布扩大 Labs 这个团队。Labs 是专门做前沿实验性产品孵化的部门,之前 Claude Code 就是从 Labs 里的研究预览版,半年时间长成了一个十亿美元级别的产品,MCP 协议现在月下载量过亿,已经成了连接 AI 和工具数据的行业标准。这次比较大的人事变化是,在 Anthropic 做了两年首席产品官的 Mike Krieger,就是 Instagram 的联合创始人,现在转去 Labs 跟 Ben Mann 一起搭团队;原来的产品组织则交给去年年底刚加入的 Ami Vora 来带,她会跟 CTO Rahul Patil 一起把 Claude 那些日活用户量很大的产品继续做大做稳。我个人觉得这次调整挺能说明 Anthropic 现在的打法,就是一边留一支小队专门在模型能力边界上折腾新东西,一边让另一支队伍专心把已经验证过的产品做扎实,两条线分得很清楚。

然后是一月十五号,这条偏案例分享类型,讲的是科学家们具体怎么用 Claude 做科研。这是去年十月 Claude for Life Sciences 发布之后的一个跟进报道,结合了他们的 AI for Science 项目,就是给全球做高价值科研的实验室提供免费 API 额度。从他们采访的实验室反馈来看,Claude 已经不只是帮着查文献、写代码这种辅助工作了,而是深度参与到实验设计该怎么做、海量数据里找规律这些环节,把原来要花几个月的项目压缩到几小时,在一些场景里甚至打开了过去做不到的研究思路。这条信息含金量也比较高,值得我们关注 AI 加科研这个方向的朋友多留意。

最后一条,一月十六号,是一条比较常规的人事公告。Anthropic 任命了 Irina Ghose 担任印度区董事总经理,她之前是微软印度的董事总经理,有三十多年的科技行业经验。这次任命是为 Anthropic 即将在班加罗尔开设的首个印度办公室做准备,算是公司全球化布局的一步,一两句话带过就行。
