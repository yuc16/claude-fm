大家好,欢迎收听 Claude FM,这里是 Anthropic 一周快讯。本期咱们盘点的是二零二六年二月八号到二月十四号这一周,这一周 Anthropic 一共有五条值得说的动态,咱们按时间顺序一条一条来。

第一条,二月十一号,Anthropic 宣布了一个挺有意思的承诺,说会承担自己数据中心给周边居民带来的电价上涨成本。背景是这样的,训练一个顶尖大模型未来需要的电力是吉瓦级别的,美国整个 AI 行业接下来几年可能需要超过五十吉瓦的电力容量,这意味着要大量新建数据中心。但问题是,数据中心接入电网往往需要新建或升级输变电设施,这部分成本通常会摊到普通居民的电费账单上,另外新增的用电需求也会推高整体电价。Anthropic 这次明确表态,会百分之百承担数据中心接网所需的电网升级费用,同时还会主动去采购新增发电产能来匹配自己的用电需求,如果新发电产能还没上线,就和电力公司及外部专家一起评估、补偿因为他们数据中心而导致的电价上涨部分。我个人觉得这个动作挺值得关注,因为它等于是 AI 巨头主动给自己的基础设施扩张承担外部成本,这种公开的电价承诺在行业里目前还不算常见,可能会成为其他 AI 公司被对比的一个参照点。

接下来,二月十二号,这周最重磅的消息来了,Anthropic 完成了三百亿美元的 G 轮融资,投后估值达到三千八百亿美元。这轮由 GIC 和 Coatue 领投,D. E. Shaw Ventures、Dragoneer、Founders Fund、ICONIQ、MGX 联合领投,参与方还包括贝莱德旗下基金、黑石、富达、高盛另类投资、摩根士丹利、淡马锡、红杉资本等一长串知名机构,微软和英伟达此前宣布的部分投资也包含在这一轮里。Anthropic 的首席财务官 Krishna Rao 提到,无论是创业者、初创公司还是大型企业客户,反馈都是 Claude 正在变得越来越关键。这笔钱会继续投入前沿研究、产品开发和基础设施扩张,简单说就是 Anthropic 现在的资金体量已经和它在企业 AI、编程领域的领先地位匹配起来了,这个估值也再次印证了资本市场对 Anthropic 商业化路径的信心。

另外这周,二月十二号还有一条,Anthropic 向一个叫 Public First Action 的组织捐款两千万美元。材料里讲的逻辑是,AI 能力进步得太快,带来巨大收益的同时风险也在累积,比如被用来自动化网络攻击,未来甚至可能涉及危险武器的研发,所以接下来几年的 AI 政策决策会影响到劳动力市场、未成年人保护、国家安全等几乎所有公共生活领域。这笔捐款就是冲着推动他们认为合理的、既能释放 AI 红利又能控制风险的政策方向去的,这条算是公司动态层面的一笔,一句话带过就好。

第四条,二月十三号,Anthropic 和 CodePath 达成合作。CodePath 是美国最大的高校计算机科学教育机构,这次合作会把 Claude 和 Claude Code 全面整合进它的课程体系,覆盖超过两万名学生,这些学生主要来自社区大学、州立学校,还有历史上以黑人为主的高校。值得一提的是,CodePath 四成以上的学生家庭年收入低于五万美元,这次合作能让他们接触到本来通常只有富裕院校学生才能用上的前沿 AI 工具和职业资源。去年秋天已经有上百名 CodePath 学生试点用 Claude Code 去给 GitLab、Puter 这些开源项目贡献代码了,反馈说即使编程语言不熟悉,Claude Code 也能帮上大忙。这条算是教育合作类的公司动态,简单提一下就行。

最后一条,二月十三号,Chris Liddell 加入了 Anthropic 的董事会。他有三十多年的高管经验,做过微软、通用汽车、国际纸业的首席财务官,还在特朗普第一个总统任期里当过白宫副幕僚长。Anthropic 联合创始人 Daniela Amodei 提到,随着 AI 对社会的影响越来越大,这种懂技术、懂公共政策、懂治理的判断力正是董事会需要的。这条也是人事任命类的动态,一句话带过。

总的来说,这周 Anthropic 的重头戏毫无疑问是那笔三百亿美元的融资,把估值直接推到三千八百亿美元,基本坐实了它在 AI 行业里的头部位置;而电价承诺这条我个人觉得也挺值得留意,因为它透露出 Anthropic 在基础设施扩张上想走一条对公众更友好的路线。好,以上就是本期 Anthropic 一周快讯,我们下期再见。
