大家好,这里是 Claude FM,欢迎收听本周的 Anthropic 一周快讯。这一期覆盖的是九月二十八号到十月四号这一周,这周一共有两条消息,而且都集中在同一天发出来的,信息量不小,我们一条一条来看。

第一条,九月二十九号,Anthropic 正式发布了新一代模型 Claude Sonnet 四点五。官方的说法是,这是目前全世界最强的编程模型,同时也是构建复杂智能体任务最强的模型,在操作电脑这件事上表现也是最好的,另外推理和数学能力相比之前也有比较大的提升。值得一提的是,官方还特别强调这是他们发布过的对齐程度最高的旗舰模型,也就是说在能力变强的同时,安全性和可控性也同步在提升,这个信号我觉得对长期关注 Anthropic 路线的人来说挺重要的,说明他们没有为了卷能力指标而牺牲对齐这条主线。围绕这次发布,Anthropic 还同步给自己的产品线做了一连串升级,包括给 Claude API 加入了上下文编辑功能和记忆工具,让智能体可以跑更久、处理更复杂的任务,还把代码执行和文件生成,比如生成表格、幻灯片、文档这些能力,直接接入到了对话里。Chrome 插件也对此前排队等候的 Max 用户开放了。

接下来第二条,同样是九月二十九号发出的,主要讲的是 Claude Code 这款产品本身的几项重要更新。最大的亮点是新增了检查点功能,这是用户呼声很高的一个需求,简单说就是 Claude Code 在帮你写代码的过程中会自动保存进度,如果某一步改动不满意,可以瞬间回滚到之前的状态,这对于让模型更自主地长时间干活来说是个关键的安全网。另外 Anthropic 还推出了原生的 VS Code 插件,可以在 IDE 侧边栏里实时看到 Claude 的改动和内联的代码差异对比,终端界面也升级到了二点零版本,增加了搜索历史命令这类体验上的改进。比较值得关注的是,他们把原来的 Claude Code SDK 正式升级改名成了 Claude Agent SDK,这意味着 Anthropic 把自己内部用来打造 Claude Code 的那套核心工具、上下文管理和权限框架,直接开放给了外部开发者去搭建自己的定制化智能体,官方提到已经有团队在用它做合规审查、网络安全、代码调试这类智能体应用了。

总的来说,这一周 Anthropic 的重心非常集中,就是用 Claude Sonnet 四点五这颗新模型,撑起了 Claude Code 整条产品线的一次大升级,从模型能力到开发工具到智能体生态,基本是一套组合拳打出来的。我个人觉得,这次发布里检查点功能和 Agent SDK 的开放,可能比模型跑分本身更值得开发者关注,因为它们直接决定了你能不能放心让 Claude 更自主地长时间干活。本周快讯就到这里,我们下期再见。
