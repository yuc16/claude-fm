## Script

大家好,欢迎收听 Claude FM,这是 Anthropic 一周快讯栏目。本期覆盖二月一号到二月七号这一周,这周一共有四条值得说的动态,我们按时间顺序一条一条来聊。

第一条,二月二号,Anthropic 宣布和艾伦研究所,还有霍华德休斯医学研究所建立合作关系,作为生命科学领域的首批合作伙伴。简单说就是,现在生物学研究产生的数据量特别大,从单细胞测序到全脑连接组学,但是把这些数据变成真正能验证的生物学洞见,中间还卡着一个大瓶颈,知识整合、假设生成这些环节还是靠人工,跟不上数据增长的速度。Anthropic 想把 Claude 的模型能力,还有它在智能体系统和可解释性上的积累,带到这两家顶级科研机构里,让科学家可以用 Claude 来规划和执行实验。这条算是公司层面的科研合作动态,了解一下就行。

接下来,二月三号,苹果的 Xcode 这边有更新。去年九月的时候 Xcode 二十六已经能用上 Claude Sonnet 四,但当时只能逐条对话式地帮忙写代码、调试。现在 Xcode 二十六点三这个版本,直接原生集成了 Claude Agent SDK,也就是驱动 Claude Code 的那套底层引擎。这意味着开发者不用离开 IDE,就能用上子智能体、后台任务、插件这些 Claude Code 的完整能力,可以让 Claude 在 Xcode 里做更复杂、更长时间的自主编码工作,而且还能直接看 Xcode 的预览界面,识别界面问题再自己迭代。这条对苹果生态里的开发者来说是个挺实在的效率提升,值得多说两句。

另外这周,二月四号,Anthropic 发了一篇态度比较鲜明的文章,表态 Claude 的对话空间会保持无广告。他们的理由是,和搜索引擎、社交媒体不一样,跟 AI 对话是一种很开放的交互形式,用户往往会分享比搜索关键词更多的上下文和个人信息,这种开放性恰恰也让对话更容易被广告这种第三方利益影响。所以 Anthropic 选择不在 Claude 的回答里插入赞助内容,也不会让广告主影响 Claude 给出的建议。这条算是一个产品原则和价值观的声明,体现了 Anthropic 想把 Claude 定位成站在用户这一边的助手,这个态度本身值得留意,但具体执行细节不多,简单提一下。

最后一条,也是这周分量最重的,二月五号,Anthropic 发布了新模型 Claude Opus 4.6,也就是现在最聪明的那个模型系列的升级版。相比上一代,Opus 4.6 在编码能力上有明显提升,规划更细致,能更长时间地维持智能体任务不掉线,在大型代码库里运行更稳定,而且代码审查和调试能力也更强,能自己抓出自己的错误。一个比较关键的突破是,这是 Opus 级模型第一次支持一百万 token 的上下文窗口,目前是测试阶段。除了编码,Opus 4.6 在金融分析、研究、文档表格演示这类日常办公场景上也表现不错,尤其是在 Cowork 这种可以多任务自主处理的场景里能派上用场。从评测数据看,它在智能体编程评测 Terminal Bench 二点零上拿了最高分,在综合推理测试 Humanity's Last Exam 上也领先所有前沿模型,在经济价值知识工作评测上,比 OpenAI 的 GPT 五点二高出大约一百四十四个 Elo 分,比自己的上一代 Opus 四点五也高出一百九十分。这条信息含量比较大,也是这周最值得开发者关注的动态。

总结一下,这一周 Anthropic 在科研合作、开发者工具集成、产品价值观表态三个方向都有动作,但真正的重头戏还是 Opus 四点六这次升级,百万 token 上下文加上编码和智能体能力的全面提升,值得花时间去试试看。本期就到这里,我们下期再见。
