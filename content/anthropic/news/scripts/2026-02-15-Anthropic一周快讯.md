大家好,欢迎收听 Claude FM,这里是 Anthropic 一周快讯。本期覆盖二月十五号到二月二十一号这一周,一共五条消息,我们按时间顺序来说。

第一条,二月十六号,Anthropic 正式开设了班加罗尔办公室,同时宣布了在印度企业、教育、农业等多个领域的新合作。这条主要是公司动态,简单说一下背景:印度是 Claude 点 ai 的第二大市场,当地接近一半的使用都跟写代码、做工程相关,所以 Anthropic 在印度加码,逻辑上是顺理成章的。

接下来,二月十七号,Anthropic 和印度的信息技术服务巨头 Infosys 宣布合作,目标是给电信、金融、制造这些监管比较严的行业做 AI agent。说白了就是把 Claude 和 Claude Code 接入 Infosys 自己的 AI 平台 Topaz,帮企业在合规和透明度的前提下用上生成式和智能体技术。这也是一条偏公司合作性质的新闻,一句话带过就行。

同一天,二月十七号,还有一条,Anthropic 和卢旺达政府签了一份三年的合作备忘录,内容覆盖医疗、教育和公共部门。比较值得一提的是,这是 Anthropic 在非洲大陆第一次通过政府层面的备忘录,把多个领域的合作正式确定下来,具体包括帮助卢旺达推进消除宫颈癌、降低疟疾和孕产妇死亡率这些医疗目标,也会给政府部门的开发者提供 Claude 和 Claude Code 的使用权限和培训。整体上还是属于公司动态类消息,不过覆盖范围比较广,值得提一句。

接下来是这周含金量最高的一条,还是二月十七号,Anthropic 发布了 Claude Sonnet 四点六。官方说这是目前最强的 Sonnet 模型,在编程、电脑操作、长上下文推理、agent 规划这些能力上做了全面升级,而且首次给 Sonnet 系列带上了一百万 token 的上下文窗口,目前是公测状态。对开发者来说更实际的变化是,早期内测的开发者反馈,Sonnet 四点六的编程能力在一致性和指令遵循上有明显提升,很多人甚至更喜欢它,而不是去年十一月发布的、级别更高的 Opus 四点五。也就是说,过去很多需要上 Opus 级别模型才能搞定的真实办公任务,现在用 Sonnet 四点六就够了,这对成本敏感的团队是个挺实际的利好。价格保持不变,还是每百万 token 三美元和十五美元这个区间。安全方面,Anthropic 也说这次的评估结果显示 Sonnet 四点六跟之前的模型一样安全,甚至更安全一些。免费版和专业版用户现在默认用的就是这个模型。

最后一条,二月二十号,Anthropic 推出了一个叫 Claude Code Security 的新功能,目前是有限的研究预览阶段,内置在网页版的 Claude Code 里。它的作用是扫描代码库找安全漏洞,然后给出针对性的补丁建议交给人来审核。这条也值得说两句:传统的静态分析工具主要靠已知模式去匹配漏洞,容易漏掉那些比较隐蔽、依赖上下文的问题,而这正是攻击者经常利用的突破口。Anthropic 之前已经证明过 Claude 能发现一些新型的高危漏洞,这次就是把这种能力直接交到防御者手里,优先开放给企业和团队版客户,开源项目的维护者还能走加速通道申请。这也是 Anthropic 应对所谓 AI 赋能攻击这种新型威胁的一个具体动作。

这周整体看下来,公司层面在印度和非洲两条线都在加速扩张,但真正值得记一下的还是 Sonnet 四点六这次升级,百万 token 上下文加上接近 Opus 级别的表现,我个人觉得这会是接下来一段时间很多团队默认切换的模型。本期快讯就到这里,我们下期再聊。
