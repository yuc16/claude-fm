## Shownotes（复制到小宇宙）

Anthropic 在二〇二六年五月正式推出了 Claude Platform on AWS,让 AWS 客户第一次能用 AWS 自己的认证和账单体系,拿到和原生 Claude API 几乎同步的全套功能。这期节目带你梳理这个新产品具体包含什么,它和老牌的 Amazon Bedrock 到底有什么区别,以及企业到底该怎么在两条路之间做选择。

本期要点:
- Claude Platform on AWS 正式全面上线,新功能和原生 Claude API 同步发布,不再有滞后
- 认证走 AWS IAM,审计走 CloudTrail,账单走 AWS 发票并可抵扣已有承诺额度
- 内含 Managed Agents、Skills、MCP 连接器、代码执行、Files API 等十余项原生功能,外加 Claude Console 开发环境
- Amazon Bedrock 依然保留,数据处理方仍是 AWS,适合严格数据驻留要求的客户
- 已有 Bedrock 私有报价的企业,务必先联系客户经理确认折扣迁移,折扣不能补溯
- 多个新功能仍处于 beta 阶段,生产环境部署建议预留灰度和回滚方案
