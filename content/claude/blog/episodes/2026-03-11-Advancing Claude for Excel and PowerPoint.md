这一期我们聊聊 Anthropic 在二〇二六年三月发布的一篇博客，主题是 Claude for Excel 和 Claude for PowerPoint 的两项重要升级：跨文件上下文共享和技能系统。这两个功能听起来不起眼,但对于天天和表格、幻灯片打交道的金融分析师、咨询顾问、运营人员来说,可能是真正能省下大量重复劳动的改动。我们会拆开讲讲这套系统是怎么设计的,Claude 给的内置技能都覆盖了哪些场景,以及普通听众该怎么上手用起来。

本期要点：
- Claude for Excel 和 Claude for PowerPoint 现在可以共享同一个对话里所有打开文件的上下文,跨应用协同工作
- 技能功能正式进入 Excel 和 PowerPoint 插件,可以把一套工作流程保存成一键复用的操作
- 官方预置了一批面向财务分析场景的起始技能,覆盖模型审计、估值建模、可比公司分析、数据清洗、行业对标 deck、改稿和审稿
- 指令功能用来设置长期生效的偏好,比如统一数字格式或幻灯片排版规则,设置一次永久生效
- Claude for Excel 和 PowerPoint 现在支持通过亚马逊 Bedrock、谷歌云 Vertex AI、微软 Foundry 三大云平台接入,方便企业合规落地
- Claude 同时也在微软三六五 Copilot 里以 Agent Mode 的形式原生运行,可以和 Copilot 并肩工作
