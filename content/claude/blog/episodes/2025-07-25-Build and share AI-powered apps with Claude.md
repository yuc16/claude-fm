## Shownotes（复制到小宇宙）

这一期我们聊聊 Anthropic 在二〇二五年七月推出的一项功能：让 Claude 直接在对话里生成可以调用 AI 能力的互动应用，并且一键分享给别人用,而你自己不用掏一分钱 API 费用。我们会拆解这个机制是怎么运作的,为什么它解决了开发者一直头疼的"做了 demo 没钱养"的问题,以及普通工程师现在可以拿它做点什么。

- Claude 的 artifacts 升级成可以反向调用 Claude API 的"活"应用,不再是静态代码片段
- 使用者用自己的 Claude 账号鉴权,消耗算在使用者自己的订阅额度上,开发者零成本
- 不需要管理 API key,不需要部署服务器,做完直接发链接就能分享
- 已经出现的真实案例:会记忆的 NPC 游戏、自适应难度的学习工具、上传 CSV 就能问答的数据分析小应用
- 目前的限制也很关键:不能调外部 API、没有持久化存储、只能用文本补全接口
- 我个人的看法:这是 Claude 在"轻量级应用分发"赛道上的一次重要试探,也是理解后续 Claude 生态演化的一个关键节点
