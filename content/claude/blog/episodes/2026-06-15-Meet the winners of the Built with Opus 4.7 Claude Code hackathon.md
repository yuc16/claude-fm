本期解读 Anthropic 于二〇二六年六月十五日发布的"Built with Opus 4.7 Claude Code 黑客松冠军揭晓"文章。六位来自土耳其、法国、智利、丹麦的参赛者，有医生、修电器工匠、CS 教授、全栈开发者，还有从未写过代码的二十岁年轻人，他们在五天内各自交付了一款完整产品——覆盖医学培训、电子维修、编程教育、互动玩具、家居维修和工厂预测维护六个垂直领域。这期节目将逐一拆解每个项目的核心设计、技术亮点，以及背后可以直接借用的开发方法论。

- Bedirhan Keskin 用 Claude 托管代理搭建医学模拟诊所 Medkit，三所伊斯坦布尔医学院将启动试点
- Alexis Chapellier 的 Wrench Board 借助 Opus 四点七的视觉理解，追踪主板电源路径诊断电子故障
- Paula Vásquez-Henríquez 的编程教育 IDE Maieutic 强制学生先写规格说明才能开始编码，对抗 AI 代写
- Rene Hangstrup Møller 的 Virtual Puppet Theater 用摄像头加语音驱动实时三维木偶互动，已 MIT 开源
- Benjamin Torralbo 把父亲三十年木工经验编成 JSON 注入模型，让 MaestrIA 诊断本地木材特定病害准确率提升七个点
- ARIA 工业预测维护系统展示 Claude 托管代理如何把五天变成可行的交付周期而非五周
