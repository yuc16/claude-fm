本期我们聊聊 Anthropic 在二〇二六年一月二十八日发布的 Claude Team 定价更新。这次更新把团队版标准席位和高级席位的价格都调低了,还给高级席位第一次加上了年付折扣,同时强调了团队版背后共享协作、知识连接器和编程代理 Claude Code 全员可用的设计逻辑。如果你的团队还在用零散的个人订阅,或者正在纠结要不要上团队版,这期内容能帮你把账算清楚。

- 标准席位降至年付每月二十美元,月付每月二十五美元
- 高级席位降至年付每月一百美元,月付每月一百二十五美元,首次拥有年付折扣
- 每个团队版席位的用量都高于 Pro 个人版,高级席位用量是标准席位的五倍
- 用量不够时,管理员可以为团队额外购买弹性容量
- 团队版自带共享工作空间、知识连接器,以及人人可用的 Claude Code 编程代理
- 默认不用客户数据训练模型,集中化管理后台让大规模团队采购更省心
