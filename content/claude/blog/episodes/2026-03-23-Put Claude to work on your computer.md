本期带你读懂 Anthropic 关于让 Claude 直接操作你电脑的最新功能。Claude 现在不仅能调用 Slack、Google 日历这样的连接器,在没有接口的地方,还能像人一样点鼠标、敲键盘、操作浏览器去完成任务,并且配合 Dispatch 派工功能,你可以用手机派活、电脑自动执行。我们聊聊这个功能背后的工具优先级逻辑、安全防护机制,以及普通用户和开发者可以怎么上手试用、要避开哪些坑。

本期要点:
- Claude 在 Claude Cowork 和 Claude Code 中新增计算机使用能力,没有连接器时可直接操作鼠标、键盘和屏幕
- 工具优先级逻辑:连接器优先,点屏幕操作只是没有接口时的兜底方案
- 多层安全机制:访问新应用需明确授权、随时可一键打断、内部扫描检测提示词注入、部分高风险应用默认禁用
- 搭配 Dispatch 派工功能,可用手机派任务、电脑自动执行,实现跨设备的异步工作流
- 目前仍是研究预览阶段,复杂任务可能需要重试,速度比原生接口集成慢,仅支持 Pro 和 Max 订阅用户的 macOS 和 Windows 桌面端
- 实操建议:从低风险任务开始练手,做好权限隔离,留意国内访问限制和工作生活边界问题
